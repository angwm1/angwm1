### 🤖 AI Trading Bot (pet project, built in Go)

A fully autonomous trading system I built to explore **quant trading, agent-based architectures, and concurrent Go systems**.

It’s made up of independent agents (market state, signals, execution, risk, evaluation, evolution) that read/write shared state and operate without human intervention.  
Instead of hard stops, risk is handled through **capital decay and reallocation**, so bad strategies fade out naturally.

I built this mainly to learn how professional trading systems are structured and how to write safe, testable Go code.

**Stack:** Go, goroutines/channels, tests, CI, event-driven architecture
